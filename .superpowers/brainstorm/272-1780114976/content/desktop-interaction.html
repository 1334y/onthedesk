窗口系统架构

Tauri 透明叠加窗口方案

窗口属性

  • decorations: false — 无边框
  • transparent: true — 透明背景
  • alwaysOnTop: true — 置顶
  • skipTaskbar: true — 不在任务栏显示
  • resizable: false
  • focus: false — 点击不抢焦点
  • 窗口尺寸 = 屏幕逻辑分辨率（全屏覆盖）

关键特性

  • 点击穿透：非角色区域鼠标穿透到下层窗口
  • 角色区域接收鼠标（单击/拖拽/右键）
  • 透明区域完全不可见、不可交互
  • 窗口跟随屏幕分辨率变化自动 resize
📐 实现方式

全屏透明窗口覆盖桌面，PixiJS Canvas 渲染在窗口上。
仅在有角色像素的位置响应鼠标事件（用 Canvas 像素级碰撞检测）。
其他区域通过 setIgnoreCursorEvents(true) 实现鼠标穿透。

备选方案（各平台差异）：
• Windows: SetWindowLongPtr + WS_EX_TRANSPARENT + 区域掩码
• macOS: NSWindow.ignoresMouseEvents + 动态切换
• Linux: X11 shape extension / Wayland 差异较大

窗口检测 & 攀爬系统

三级窗口感知

🪟 L1: 窗口枚举

通过 OS API 获取所有顶层窗口：

  • Windows: EnumWindows / GetWindowRect
  • macOS: CGWindowListCopyWindowInfo
  • Linux: X11 XQueryTree / _NET_CLIENT_LIST

过滤：排除自身、排除最小化/不可见窗口、排除全屏窗口

📐 L2: 边缘提取

从窗口矩形计算可攀爬边缘：

  • 顶部边缘 → 可行走/可坐下
  • 左右边缘 → 可垂直攀爬
  • 底部边缘 → 可悬挂
  • 边缘段校验：排除被其他窗口遮挡的部分

🎯 L3: 攀爬决策

行为仲裁器决定是否攀爬：

  • 附近有窗口边缘 → 概率走高
  • 体力和心情影响攀爬意愿
  • 攀爬路径 A* 寻路
  • 边缘坐下概率与心情正相关
🪜 攀爬流程
1. 检测到窗口边缘在角色附近（阈值 20px）
2. 行为仲裁 → 决定攀爬
3. 边缘接触点锚定 → WALK → 贴边 → CLIMB 动画 → 到达顶部边缘
4. 顶部行走/SIT/继续攀爬更高窗口
5. 边缘消失（窗口移动/关闭）→ FALL 动画 → 落到下一表面或地面

物理 & 移动系统

地面检测 & 重力
  • 地面 = 任务栏顶部 / 屏幕底部
  • 平台 = 所有窗口顶部边缘
  • 站在地面上：角色 y = ground_y - height
  • 站在窗口上：角色 y = window_top - height
  • 无地面支撑 → 进入 FALL 状态
  • FALL → 射线向下检测第一个表面 → 着陆
物理简化：角色始终站在某表面或下落中
不模拟惯性/动量（像素风不需要）
行走移动模型
  • 随机游走：选方向 → 走 50~200px → 停顿
  • 目标导向：走向窗口边缘 / 任务栏区域
  • 水平速度：~60 px/s（2x 缩放后）
  • 方向切换：走到边缘或随机转向
  • 任务栏活动：沿任务栏横向行走，跳跃图标
PixiJS Ticker 驱动位置更新
每帧: 位置 += 速度 × delta

多平台窗口 API 差异

平台适配层设计
能力 Windows macOS Linux (X11)
透明置顶 Win32 API + Tauri NSWindow level _NET_WM_STATE_ABOVE
鼠标穿透 WS_EX_TRANSPARENT + 区域掩码 ignoresMouseEvents XShapeCombineRectangles
窗口枚举 EnumWindows CGWindowList XQueryTree + _NET_CLIENT_LIST
任务栏检测 FindWindow("Shell_TrayWnd") Dock 检测较难* _NET_WM_STRUT 属性
活动窗口检测 GetForegroundWindow NSWorkspace.frontmostApplication _NET_ACTIVE_WINDOW
系统托盘 Shell_NotifyIcon NSStatusBar AppIndicator / libappindicator
* macOS Dock 检测有沙箱限制，需辅助功能权限。优先实现 Windows/Linux 任务栏行为，macOS 降级为屏幕底部区域活动。
平台差异封装在 Tauri Rust 后端的一个 PlatformProvider trait 中，编译时条件编译。
